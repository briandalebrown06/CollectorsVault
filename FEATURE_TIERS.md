# # Movie Collectors
## Feature Tiers

This file defines the locked feature breakdown for the Movie Collectors app by tier.

---

# Free Tier

The Free tier should provide a strong, usable collector app on its own.

## Library basics

- Add movies manually
- Add editions manually
- Edit entries
- Delete entries
- Browse in list view
- Browse in grid view

## Collection status

- Owned
- Wishlist
- Preorder
- On Order
- Delivered
- Sold / Removed

## Watch / open status

- Sealed
- Open, not watched
- Open and watched

## Condition tracking

- Mint
- Very Good
- Good
- Fair
- Damaged
- Damage notes

## Purchase / order details

- Retailer, manual
- Tracking number, manual
- Purchase notes
- Shipping status, manual

## Organization

- Tags
- Custom lists

## Theme / settings

- Dark mode
- Light mode
- System mode

## Backup

- Export backup locally

## Artwork

- Choose main display art per item
- Support poster art
- Support release cover art
- Support user-uploaded art or photo as primary image

# Collector Pro
## One-time purchase

Collector Pro adds advanced local-first collector features.

## Library intelligence

- Barcode scanning
- Advanced filters
- Saved smart filters
- Duplicate detection
- Upgrade warnings
- Completeness score per item

## Edition / release structure

- Box set support
- Parent-child edition support
- Multiple editions per title
- Title hub with edition selector
- Better franchise / series grouping
- Better label grouping

## Physical collection features

- Shelf location mapping
- Prettier shelf browsing
- More custom fields
- Custom retailer creation
- Retailer-aware preorder cards

## Import / export

- Better CSV export
- Better CSV import
- Easier migration from another library/app
- Field-mapping import wizard
- Duplicate merge review
- Import preview / health report

## Social / profile

- Public profile link
- Collection comparison
- Social reactions
- Emoji reactions
- Public/private collection controls
- Shareable custom lists

## Artwork / image system

- Multiple images per item
- Artwork gallery
- Reorder images
- Image type tagging
- Alternate artwork support
- User-uploaded photo of own copy
- Steelbook art
- Slipcover photos
- Back cover photo
- Spine photo
- Disc art photo

## Slipcover support

- Slipcover yes/no
- Original slipcover yes/no
- Replacement/custom slipcover
- Slipcover condition
- Slipcover notes
- Slipcover photos
- Filter by with-slip / no-slip

## Disc swap / custom configuration

- Mark disc swaps
- Mark release A in release B shell
- Case-from field
- Disc-from field
- Custom configuration notes
- Visible disc-swap badge

## Upgrade intelligence

- Flag direct upgrades:
  - Blu-ray to 4K
  - DVD to Blu-ray
  - 5.1 to Atmos
  - HDR to Dolby Vision
  - similar quality-jump paths
- Owned but inferior filter
- Owned and optimal badge
- Upgrade-loses-features warning
- Compare owned edition to better edition

## Best release logic

- Best overall release
- Best budget release
- Best extras release
- Best packaging release
- Best audio/video release
